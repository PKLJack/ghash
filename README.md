# GHash

## What it does

A program for calculating common checksums,

![GHash](docs/images/ghash-demo-01.png)

![GHash](docs/images/ghash-demo-01.webm)

Algorithms:

- sha1sum
- sha2sum
- sha2sum
- sha3sum
- sha5sum
- md5
- crc32

## Usage

```sh
# Build with poetry
poetry build

# Install with pipx
pipx install ./dist/ghash-X.X.X.whl
```
